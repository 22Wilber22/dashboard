Proyecto LIDO Dashboard

Pasos rápidos para usar el scaffold Vite creado:

1) Instalar dependencias

```bash
npm install
```

2) Ejecutar en desarrollo

```bash
npm run dev
```

3) Build para producción

```bash
npm run build
npm run preview
```

Notas:
- El HTML original está en `lido_dashboard_rediseno.html`.
- `index.html` carga `src/main.js` (migración del script inline).
- El proyecto usa `chart.js` y `vite`.
